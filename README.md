# Market Tracker - Stocks & Crypto

A comprehensive real-time market tracker displaying the top 50 Nasdaq stocks and top 50 cryptocurrencies with live market data.

## Features

### Stocks Section
- Real-time stock prices from Yahoo Finance API
- Top 50 Nasdaq stocks by market cap
- Market capitalization display
- 24-hour price change indicators with percentages
- Live price updates

### Cryptocurrency Section
- Real-time cryptocurrency prices from CoinGecko API
- Top 50 cryptocurrencies by market cap
- Market Cap and FDV (Fully Diluted Valuation) display
- 24-hour price change indicators
- Crypto icons for visual identification
- Support for high-precision decimal prices

### General Features
- Auto-refresh every 60 seconds
- Manual refresh button for both stocks and crypto
- Responsive grid layout
- Clean, modern UI with purple gradient background
- Separate sections for stocks and cryptocurrencies

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Yahoo Finance API (for stocks)
- CoinGecko API (for cryptocurrencies)
