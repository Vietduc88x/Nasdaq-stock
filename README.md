# Top 50 Nasdaq Stocks Tracker

A real-time stock price tracker displaying the top 50 Nasdaq stocks with live market data.

## Features

- Real-time stock prices from Yahoo Finance API
- Market capitalization display
- Price change indicators with percentages
- Auto-refresh every 60 seconds
- Responsive grid layout
- Manual refresh button

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Yahoo Finance API
